# Insurance complaint & litigation rates — data collection

A growing collection of publicly available, machine-readable datasets that quantify "bad behavior by insurance companies" — complaints, lawsuits, regulator findings, statutory notices — across U.S. states and over time. Each dataset lives in its own subfolder with the scripts, source provenance, and caveats needed to reproduce or audit it independently.

## Datasets in this repo

| Folder | What | Coverage | Status |
|---|---|---|---|
| [`fjc_idb/`](fjc_idb/README.md) | Federal civil filings coded `Nature of Suit = 110` (Insurance), by state and calendar year. The only nationwide longitudinal lens on insurance litigation pressure. Includes an origin-code breakout (original vs removed-from-state-court). | All 50 states + DC + 4 territories, 1988–present | Built 2026-05-04 |
| [`fjc_idb/viz/`](fjc_idb/viz/README.md) | Interactive timeline viz over the FJC dataset above — single-file HTML, opens in any browser. Pan/zoom/reset, multi-select states, per-series average + moving-average overlays, cross-state aggregates. | (same coverage) | Built 2026-05-04 |
| [`viz/`](viz/README.md) | **Unified data viewer** — single HTML covering every dataset in this repo. Picker grouped by category (regulator findings / plaintiff allegations / federal lawsuits), with per-dataset tailored controls and caveat banners. Manifest-driven so future states drop in without code changes. | All datasets above | Built 2026-05-04 |
| [`tx_tdi/`](tx_tdi/README.md) | Texas Department of Insurance closed consumer complaints. TDI's own `Confirmed`/`Not Confirmed` finding per complaint — the closest TX analogue to MD MIA's "in favor of insured" series. | TX, 2012–present | Built 2026-05-04 |
| [`ca_cdi/`](ca_cdi/README.md) | California Department of Insurance complaint volume + per-line per-company `Justified Complaint` counts. Parsed from CDI Annual Reports + Consumer Complaint Studies (PDF; CDI exposes no API). | CA, 2020–2024 | Built 2026-05-04 |
| [`fl_crn/`](fl_crn/README.md) | Florida Civil Remedy Notice yearly counts by line of insurance. **Plaintiff-side allegations**, not regulator findings — *measures litigation pressure, not insurer wrongdoing*. Per-filing detail deferred to a public records request. | FL, 2014–present | Built 2026-05-04 |
| [`ny_dfs/`](ny_dfs/README.md) | NY DFS upheld complaint counts. Auto via Open Data NY (per-company, 2009–2024, 2-year-rolling metric). Health via Consumer Guide PDFs (per-plan, by HMO/EPO-PPO/Commercial, 2015–2024). | NY, 2009–2024 | Built 2026-05-04 |
| [`wa_oic/`](wa_oic/README.md) | Two streams: (1) IFCA per-notice records (plaintiff-side, 2025–2026 only); (2) OIC Annual Report consumer-complaint scalars (regulator-side, 2020–2024). | WA, 2020–2026 | Built 2026-05-04 |
| [`naic_idrr/`](naic_idrr/README.md) | NAIC Insurance Department Resources Report Vol 1 — per-state consumer complaints + inquiries received by each state insurance department. The nationwide regulator-side complement to FJC IDB. Parsed from annual NAIC PDFs. | All 50 states + DC + 5 territories, 1998–2022 | Built 2026-05-04 |
| [`md_mia/`](md_mia/README.md) | Maryland Insurance Administration §27-1001 first-party bad-faith complaint dispositions. Hand-curated from 18 MIA annual reports. The cleanest 4-bucket regulator-finding dataset in the project (against / mixed / for / no-decision); serves as the model for the unified viewer's outcome breakout. | MD, FY 2008–FY 2025 | Built 2026-05-04 |
| [`ct_cid/`](ct_cid/README.md) | Connecticut Insurance Department closed consumer complaints. Per-complaint Socrata feed; CID's own `disposition` mapped to a 3-bucket against/for/ambiguous classification — CT's analogue to TX `Confirmed`/MD `in favor of insured`. | CT, 2018–2026 (effective coverage 2022+) | Built 2026-05-04 |
| [`in_idoi/`](in_idoi/README.md) | Indiana DOI annual *Company Complaint Index* — per-company by line (Annuity / Auto / Health / Homeowners / Life). NAIC-tradition complaint index = (share of complaints) / (share of premium). **Different metric class** than upheld/confirmed counts; viz keeps it in its own category. | IN, 2009–2024 | Built 2026-05-04 |
| [`ks_kid/`](ks_kid/README.md) | Kansas DOI annual *Complaint Index Report* — per-company by line for top-20 premium writers + 10+-complaint companies. Same NAIC-tradition complaint-index metric as IN; different inclusion rules (KS includes zero-complaint top-20 writers). | KS, 2020–2024 | Built 2026-05-04 |
| [`bad_faith_rank/`](bad_faith_rank/README.md) | **State bad-faith protection ranking.** Composite ranking of all 51 jurisdictions on 11 weighted factors (statutory PRoA, common-law tort, damages, fee shifting, admin remedy, etc.) synthesized from UPH/Feinman 2025, NAIC MC-55, IADC and Wilson Elser 50-state surveys. Single-file HTML viewer with **live weight sliders** + per-state factor breakdowns with citations. Both score-band tiers and structural clusters defined. | All 50 states + DC | Built 2026-05-04 |
| [`va_scc/`](va_scc/README.md) | Virginia State Corporation Commission, Bureau of Insurance — annual workload by line (P&C / L&H) and a separate health External Review disposition table mapped to the canonical 4-bucket outcome taxonomy. Only Phase 3 candidate that passed recon. | VA, FY 2022–FY 2025 | Built 2026-05-04 |
| [`id_doi/`](id_doi/README.md) | Idaho DOI Consumer Complaint Comparison Tables — per-company complaint index by line (auto / homeowners / group health / individual health), HTML scrape. Same NAIC-tradition metric class as IN/KS. | ID, 2018–2020 | Built 2026-05-04 |
| [`il_idoi/`](il_idoi/README.md) | Illinois IDOI Consumer Complaint Ratio Reports — per-company by line, dual-layout PDF parser. IL changed ratio definition between 2018 (NAIC share-of-share) and 2019+ (per $1M EP for P&C, per 10k policies for Life). 2,671 per-company rows. | IL, 2018–2024 (no 2021/2022) | Built 2026-05-04 |
| [`co_doi/`](co_doi/README.md) | Colorado DOI Annual Complaint and Recoveries Reports — workload by line (P&C / L&H / homeowners / health) plus per-line $ recovered for consumers. Hand-verified inline data. CO publishes per-company indexes via separate Oracle-backed tool not in this dataset. | CO, FY 2022–FY 2025 | Built 2026-05-04 |
| [`wi_oci/`](wi_oci/README.md) | Wisconsin OCI Insurance Report — Table II per-line complaint counts (11 sub-lines + 3 totals + grand total) extracted from the annual WIR PDFs. Workload signal; per-company data lives in a separate interactive tool. | WI, 2019–2024 | Built 2026-05-04 |
| [`mo_dci/`](mo_dci/README.md) | Missouri DCI Complaint Index Reports — per-company indices (3-year-pooled per report) for 7 lines, plus per-year per-line aggregate counts and "% resolved with consumer relief" rate. NAIC-tradition methodology; closest peer to IN/KS/ID/IL with bonus per-year resolution rate. PRR was originally planned but recon found the data is published as PDFs. | MO, 2017–2023 | Built 2026-05-05 |
| [`mi_difs/`](mi_difs/README.md) | Michigan DIFS public *Complaint Statistics and Ratios* — per-company complaint ratios per $1M premium for 5 lines (Auto, Home, Life, A&H, Annuity); plus per-line aggregate counts (7 lines), per-entity-type totals (Insurance Co / HMO / Other), and per-reason × per-entity-type breakouts. Different ratio formula than MO/IN/KS. PRR was originally planned but recon found the data is published on per-line HTML pages. | MI, 2022–2024 | Built 2026-05-05 |
| [`la_ldi/`](la_ldi/README.md) | Louisiana LDI *Consumer Complaint Data* — per-company NAIC-tradition complaint indices for 4 personal lines (Auto, Homeowners, Life & Annuity, A&H), full long tail. **Deepest history in the per-company-index batch (10 years).** Captures the post-Hurricane-Ida 2021 homeowners complaint surge with cleanly-identified failed-carrier signal. Cloudflare-protected origin requires `cloudscraper` to fetch. Originally a Phase 5 candidate; passed recon. | LA, 2015–2024 | Built 2026-05-05 |
| [`or_dfr/`](or_dfr/README.md) | Oregon DFR Insurance Complaint Reports — per-company per-line per-year **with Confirmed Complaints** (outcome dimension). 6 personal lines × 7 years. **First state in the per-company-index batch with a true regulator-issued merits-decision count alongside per-company indices.** Originally a Phase 5 candidate; passed recon. | OR, 2019–2025 | Built 2026-05-05 |
| [`naic_idrr/output/idrr_complaints_yearly.parquet`](naic_idrr/output/idrr_complaints_yearly.parquet) | NAIC IDRR data republished in the cross-state aggregate schema (`state, year, line, outcome_category, count, source_type, is_partial_year, notes`) for stackability with state-specific aggregate outputs. Source: same as `naic_idrr/`; territories dropped. | 50 states + DC, 1998–2022 (no 2003) | Built 2026-05-05 |

The remaining ~30 jurisdictions are NAIC-only by design and documented in [tail_states.md](tail_states.md).

## Re-running anything

Each subfolder is self-contained — its own README explains how to fetch, build, and verify. The scripts are plain Python; the only third-party dependency for the FJC pipeline is `polars`. Source files are downloaded fresh from the canonical URL with SHA-256 manifests, so any run can be audited against the recorded provenance.
