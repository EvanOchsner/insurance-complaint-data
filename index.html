U.S. Insurance Complaint Data

Normalized complaint, lawsuit, and regulatory-finding data from 20+ state insurance regulators — with interactive viewers, reproducible scripts, and open data.

Interactive Viewers

Unified Data Viewer →

All datasets in a single interface — complaint counts, confirmed findings, litigation rates, and regulatory workload across states and years.

Federal Insurance Lawsuits →

Federal civil filings coded "Insurance" (NOS 110) by state and year, 1988–present. The only nationwide longitudinal litigation lens in the dataset.

State Bad-Faith Rankings →

Composite ranking of all 51 jurisdictions on 11 weighted factors — live sliders, per-state breakdowns with citations, and structural clusters.

About This Dataset

Each source lives in its own subfolder with the collection scripts, a SHA-256 manifest of downloaded files, a methodology note, and a provenance document recording exactly where the data came from and how it was transformed. The outputs are plain CSV and Parquet — no proprietary formats, no accounts required.

Coverage spans 20+ states, the NAIC national dataset (1998–2022), and federal court filings (1988–present). See the GitHub repository for full dataset documentation.
